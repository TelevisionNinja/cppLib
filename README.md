# cppLib
A collection of some c++ functions

Subjects that this library covers:
- arrays & vectors
- math
- strings

Arrays & Vectors:
- utilities to manipulate arrays & array lists
- search

Math:
- general math funcitons

Strings:
- utilities to manipulate strings
- search

Data Structures:
- binary heap
- doubly linked list

# Build and Run Tests
```bash
cd ./tests
g++ -std=c++23 -Wall ./*.cpp ../src/*.cpp -o test.exe
./test.exe
```
